# GitHub Profile 配置

替换以下占位符为真实信息：

## 需要替换的内容

1. `your-username` → 你的 GitHub 用户名
2. `你的邮箱` → 你的邮箱地址

## 如何使用

1. 创建一个名为 `your-username.github.io` 的仓库（必须是你的用户名）
2. 将 `README.md` 内容复制到仓库的 README.md
3. 推送到 GitHub 即可在个人主页显示

## 自定义选项

- 可以添加更多.shields.io 徽章
- 可以添加 WakaTime 统计
- 可以添加更多项目展示
- 可以自定义颜色主题
